## 🧠 Архитектура

Вход (5) → Скрытый (8) → Выход (3)
↑                       ↑
HP, враг, защита      Атака, Тяж, Защита


- **Q-обучение с подкреплением**
- **Replay Buffer:** 500 примеров
- **Epsilon-стратегия:** 30% → 5%
- **Активация:** ReLU с gradient clipping
- **Оптимизатор:** SGD с моментумом
- **Параметров:** ~75
- **Размер модели:** < 5 КБ

- Python 3
- Pydroid 3 (Android) или любой Python-интерпретатор
- Никаких внешних библиотек (только math, random, json)
